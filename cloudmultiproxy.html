🌐 CloudMultiProxy

CloudMultiProxy es un core diseñado para Velocity que permite la sincronización de infinitos proxies. Su objetivo principal es permitir escalar tu servidor, da igual cuantos jugadores se conecten.

Módulos

El plugin está dividido en varios módulos que pueden activarse o desactivarse según las necesidades de la network.

Módulo Principal

Incluye la lógica core y las funcionalidades esenciales para la gestión de la red. Permite realizar alertas globales (con filtros por proxy).

Placeholders

Integración nativa con PlaceholderAPI . Expone variables sobre el número de jugadores globales o de servidores específicos, ideal para mostrar estadísticas en hologramas, scoreboards o en el TAB.

Custom Commands

Este módulo ofrece versatilidad para crear comandos personalizados simples. Es ideal para crear atajos a enlaces o información (como tiendas, redes sociales, etc.), permitiendo configurar el mensaje y los permisos necesarios.

Lobby

Gestión inteligente de los servidores de entrada. Su principal función es el autobalanceo, redirigiendo a los jugadores al lobby con menos carga.

MOTD

Gestión dinámica MOTD en la lista de servidores. Permite la rotación de múltiples mensajes, configuración de slots y uso de placeholders para contadores y variables personalizadas.

Configuración y Flexibilidad

Uno de los puntos fuertes de CloudMultiProxy es su capacidad de personalización: ● Modularidad: Todos los módulos (excepto el principal) pueden habilitarse o deshabilitarse individualmente. ● Configuración Total: Prácticamente el 100% del plugin es configurable. Esto incluye todos los mensajes mostrados a los jugadores y la inmensa mayoría de las opcionesde comportamiento de cada módulo.
